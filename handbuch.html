Web-App (Desktop & Android)
BlockCalc ist eine Web-Applikation zur Standsicherheitsberechnung von tief eingespannten Blockfundamenten für Fahrleitungsmasten. Die Berechnungen erfolgen nach dem analytischen Verfahren von Steckner (1989) in Verbindung mit den Normen SIA 267:2013 (Geotechnik) und SIA 261:2020 (Einwirkungen).
Das Programm läuft vollständig im Browser — ohne Installation, ohne Serververbindung. Alle Projektdaten werden lokal im Gerät gespeichert (IndexedDB).
SIA 267:2013 SIA 261:2020 Steckner 1989 Three.js r128 Single-File App

3 · Berechnungs- & Nachweisgrundlagen
3.1 · Systemmodell Blockfundament
3D-Modell des Fundaments
aus App-Viewer exportiert
Das Blockfundament ist ein tief eingebetteter Betonkörper (Rechteck-Querschnitt B × L), der den Mastfuss aufnimmt. Unter Biegebelastung dreht sich der Block um eine Nulllinie y_NL innerhalb der Einbindungstiefe.
Oberhalb der Nulllinie wirken destabilisierende Erddrücke (aktiv Ea, mobilisierter Erdwiderstand Ew). Unterhalb der Nulllinie wirken stabilisierende Kräfte (passiver Erdwiderstand Ep, Ruhedruck E0'). An der Sohle wirken Normalreaktion N_u und Sohlreibung R_u.
Nachweiskonzept Blockfundament:
GrenzzustandKriterium
GZT Standsicherheitη = M_u / M_d ≥ 1.0  (Steckner Gl.66)
GZG Gebrauchstauglichkeittan(α) ≤ tan(α)_zul  (Winkler)
3.2 · Bemessungsgrössen & Schnittgrössen
Einbindetiefe & Querschnittsbreite
t = H_Block + ü                   [m] Effektive Einbindetiefe (Blockhöhe + Überdeckung)
b_id = B + 2 · 0.3·t · tan(φ_d)   [m] Wirksame Querschnittsbreite nach Steckner
Eigengewicht & Normalkraft
G_k = (B·L·H_Block + B_k·L_k·(H_Kopf+ü)) · γ_Beton + (B·L − B_k·L_k) · ü · γ_Auff.
N_d = γ_G · (N_G + G_k)                       [kN] Bemessungsnormalkraft
Bemessungsmoment am OK Fundament
arm = t + H_Kopf                             [m] Hebelarm für Horizontalkräfte
M_xd = H_xd·arm + γ_G·M_Gx + γ_W·M_Wx + N_d·e_y  [kNm]
M_yd = H_yd·arm + γ_G·M_Gy + γ_W·M_Wy + N_d·e_x  [kNm]
M_d = sqrt(M_xd² + M_yd²)                     [kNm] Massgebendes Bemessungsmoment
Das massgebende Bemessungsmoment M_d wird am Fundament-OK (Blockoberkante = GOK − ü) ausgewertet. Bei mehreren aktiven Richtungen wird die vektorielle Resultierende verwendet.

3.3 · GZT — Standsicherheitsnachweis nach Steckner (1989)
Das Verfahren nach Steckner (Bautechnik 2/1989) bestimmt das Grenzmoment M_u, das der eingebettete Block dem Bemessungsmoment M_d entgegenstellen kann. Der Block dreht sich um eine Nulllinie y_NL (Drehpunkt, gemessen von UK). Oberhalb widersteht der Boden passiv, unterhalb kehren sich die Druckfiguren um.
Berechnungsablauf (Steckner Bild 5): Beiwerte → b_id → Erddrücke → Reibung → y_NL → Sohlkräfte → M_u → η
3.3.1 · Erddruckbeiwerte (Gl.27–30)
Alle Erddrücke verwenden die horizontale Komponente des Coulomb-Erddrucks. Der Wandreibungswinkel δ₁ gilt für aktiven und passiv-unter-NL Druck; für passiv-über-NL wird δ_p separat bestimmt.
Wandreibungswinkel
δ₁ = φ'_k / 2      Wandreibung aktiv (positiv) + passiv unter NL  [nach Steckner §5.3]
δ_p = δ₁         Wandreibung passiv über NL (Startannahme); Reduktion wenn N_u < 0  (Gl.49/50)
λ_a — aktiver Erddruckbeiwert (Gl.27, Coulomb mit δ=+δ₁, β_a=0)
λ_a = [ cos(φ'_k) / ( 1 + √( sin(φ'_k+δ₁)·sin(φ'_k) / cos(δ₁) ) ) ]²
Beispiel φ=30°, δ₁=15°: λ_a = [0.866/(1+0.604)]² = 0.291
λ_p — passiver Erddruckbeiwert über NL (Gl.28, Coulomb mit δ=−δ_p, β_p=0)
λ_p = [ cos(φ'_k) / ( 1 − √( sin(φ'_k+δ_p)·sin(φ'_k) / cos(δ_p) ) ) ]²
Beispiel φ=30°, δ_p=15°: λ_p = [0.866/(1−0.604)]² = 4.80
Hinweis: Steckner §5.3 verwendet FEM-Ansatz δ_p = −0.5·φ'_k (neg. Wandreibung passiv)
λ'_p — passiver Erddruckbeiwert unter NL (Gl.29, mit δ=+δ₁)
λ'_p = [ cos(φ'_k) / ( 1 − √( sin(φ'_k−δ₁)·sin(φ'_k) / cos(δ₁) ) ) ]²
Beispiel φ=30°, δ₁=15°: λ'_p = [0.866/0.634]² = 1.867
λ_0 — Erdruhedruckbeiwert (Gl.30)
λ_0 = 1 − sin(φ'_k)      Jaky-Formel
Beispiel φ=30°: λ_0 = 1 − 0.5 = 0.50
Ideelle Druckwandbreite b_id (Gl.23–26)
b_id = B + 0.6 · t · tan(φ'_k)      [m] wirksame Querschnittsbreite nach Steckner
Beispiel B=1.20m, t=3.0m, φ=30°: b_id = 1.20 + 0.6·3.0·0.577 = 2.239 m
3.3.2 · Erddruckkräfte (Gl.31–35)
Alle Kräfte wirken auf die Fundamentoberfläche. Der Erddruck steigt linear ab GOK mit der Tiefe z. Das Integral umfasst die jeweilige Zone (über oder unter NL). Für Mehrschichtböden erfolgt die Integration schichtweise.
Über der Nulllinie (Höhe h = t − y)
Ea = ½·λ_a·γ_e·b_id·h²     (Gl.31)
aktiver Erddruck (destabilisierend)
Ep = ½·λ_p·γ_e·b_id·h²     (Gl.32)
voller passiver Widerstand
E0 = ½·λ_0·γ_e·A·h²       (Gl.34)
Erdruhedruck (A = B·L Grundfläche)
Unter der Nulllinie (Höhe y)
E'p = ½·λ'_p·γ_e·b_id·y(2t−y)  (Gl.33)
passiver Widerstand (Gegenseite)
E'0 = ½·λ_0·γ_e·A·y(2t−y)    (Gl.35)
Erdruhedruck unter NL
Hebelarme ab OK Fundament (Gl.36–37)
v = 2/3·(t−y) − ü                            (Gl.36) Hebelarm Kräfte über NL
Beispiel t=3.0, y=0.221, ü=0.80: v = 1.853 − 0.80 = 1.053 m
v' = (2/3)·[3t(t−y)+y²]/(2t−y) − ü    (Gl.37) Hebelarm Kräfte unter NL
Beispiel: v' = 2.890 − 0.80 = 2.091 m

3.3.3 · Reibungskräfte und resultierender Erdwiderstand (Gl.38–44)
Den Erddruckkräften zugeordnete vertikale Reibungskomponenten (tangential zur Wandfläche) beeinflussen die Sohlnormalkraft N_u und das Grenzmoment.
Reibungskräfte (Gl.38–42)
Ra = Ea · tan(δ₁)   (Gl.38) ↓ abwärts
Rp = Ep · tan(δ_p)  (Gl.39) ↑ aufwärts wenn δ_p>0
R'p = E'p · tan(δ₁)   (Gl.40) ↑ aufwärts
R0 = E0 · tan(δ₁)   (Gl.41) zugehörige Reibung
R'0 = E'0 · tan(δ₁)   (Gl.42) zugehörige Reibung
Reibungshebelarm = B/2 = A/2 (halbe Fundamentbreite), da Reibung an der Sohle angreift.
Resultierender Erdwiderstand (Gl.43–44)
Ew = Ep − Ea + R0         (Gl.43)
netto über NL, Hebelarm v
E'w = E'p + R'0             (Gl.44)
netto unter NL, Hebelarm v'
mob = 0.25: Mobilisierungsgrad passiver Erdwiderstand (Steckner §5.4). Wird nicht direkt in Gl.43 eingesetzt — Gl.43 verwendet den vollen Ep-Wert. mob erscheint im GZT-Diagramm als Visualisierung des mobilisierten Anteils zwischen E0 und Ep.
3.3.4 · Nulllinie y_NL — Bisektionsverfahren (Gl.54)
Die Nulllinie y (Abstand von UK) erfüllt die horizontale Kräftegleichgewichtsbedingung (Steckner Gl.54):
H* − Ew(y) + E'w(y) + R_u(y) = 0    (Gl.54)
→ Ew(y) − E'w(y) = H* + R_u(y)    (Bisektionsziel)
Da R_u von N_u abhängt und N_u von y, wird zweistufig iteriert:
1Näherungs-N_u mit vollem Ep über t berechnen → R_u_approx
↓
2Bisektion (50 Iter.): y₁ aus Ew(y)−E'w(y) = H*+R_u_approx
↓
3Exaktes R_u bei y₁ berechnen
↓
4Bisektion (50 Iter.): y_NL verfeinert mit exaktem R_u
Steckner leitet für den einfachen Fall (homogener Boden, kein ü) eine analytische Lösung via Hilfsgrössen U, W, S, T her (Gl.57–63). Die App verwendet das numerische Bisektionsverfahren — allgemeiner und für Mehrschichtböden geeignet.
3.3.5 · Sohlkräfte (Gl.46, 48, 51)
Sohlnormalkraft N_u (Gl.46)
N_u = N_d + Ra − Rp + R'p
N_d = γ_G·(N_G+G_k)  Bemessungsnormalkraft
Ra = Ea·tan(δ₁) ↓ (erhöht N_u)
Rp = Ep·tan(δ_p) ↑ (reduziert N_u)
R'p = E'p·tan(δ₁) ↑ (reduziert N_u)
Bedingung: N_u ≥ 0  (Gl.49)
Falls N_u < 0: δ_p stufenweise reduzieren (bis δ_p=−δ₁)
Exzentrizität u (Gl.47/48, Näherung)
u ≈ B/2 · (1 − (y_NL/B)²)
Winkler-Näherung (Abweichung <3%)
Exakt (Steckner Gl.47–48):
x = N_u / (B·p_u)   u = (A−x)/2
p_u aus Grundbruchformeln Gl.5–7
Sohlreibung R_u (Gl.51)
R_u = N_u · tan(φ'_k)
φ'_k = charakteristischer Reibungswinkel (Gl.51)
3.3.6 · Grenzmoment M_u — Steckner Gl.66
Das Grenzmoment M_u folgt aus dem Momentengleichgewicht ΣM = 0 um den Angriffspunkt von H* (= OK Block). Es enthält fünf stabilisierende Beiträge:
M_u = − (Ew_d − Ea_d + E0_a_d) · v    Gl.66 Term 1: Netto-Erdwiderstand über NL
+ (Ep_b_d + E0_b_d) · v'            Gl.66 Term 2: Kräfte unter NL (stabilisierend)
+ (Ra + Rp + R'p) · B/2            Gl.66 Term 3: Reibungskräfte × B/2
+ R_u · (t − ü)                      Gl.66 Term 4: Sohlreibung × Blockhöhe H_Block
+ N_u · u                              Gl.66 Term 5: Sohlreaktion × Exzentrizität
TermBedeutungVorzeichenHinweis
−(Ew−Ea+E0a)·vDestab. Nettokraft über NL−Bewirkt Rotation
+(Ep_b+E0_b)·v'Stabilisierend unter NL+Gegendruck
+(Ra+Rp+R'p)·B/2Reibungskräfte+Vertikale Wandreibung
+R_u·(t−ü)Sohlreibung × H_Block+Nicht R_u·t!
+N_u·uSohlreaktion+Winkler-Exzentrizität
Nachweis
η_GZT = M_u / M_d   ≥ 1.0    (Grenzzustand der Tragfähigkeit erfüllt)
M_d = H_d·(H_Kopf+ü) + γ_G·M_G + γ_W·M_W   (Bemessungsmoment am OK Block)

3.4 · GZG — Gebrauchstauglichkeit nach Steckner Gl.3 und Gl.11–21
Der GZG-Nachweis prüft die Schiefstellung des Mastfusses unter charakteristischen Lasten. Das Verfahren basiert auf den Steckner-Gleichungen 11–21 (Bautechnik 2/1989): Das Fundament rotiert starr; die Widerstände M₁ (stirnseitig) und M₂ (Sohle) begrenzen das aufnehmbare Moment M_α.
3.4.1 · Bettungsziffer C₁ (Steckner Gl.3)
C₁ = {4 + 0.6·(φ'_k − 20)} · 10⁴ · t/2    [kN/m³] (Gl.3, horizontales Gelände)
C₁ = {4 + 0.6·(φ'_k − 20)} · 10⁴ · t/2 · (1 + β_p/40)  (Gl.4, β_p ≤ 0)
Beispiel φ=30°, t=3.0m: C₁ = 10·10⁴·1.5 = 150'000 kN/m³ = 150 MN/m³
Mehrschicht: tiefengewichteter Mittelwert φ_avg; β_p = Geländeneigung
3.4.2 · Charakteristische Einwirkungen und Schnittgrössen
N_α = N_k = G_k + N_G + N_W                [kN] charakteristisch (γ = 1.0)
H_k = H_Gx + H_Wx    (Quer)  oder  H_Gy + H_Wy [kN] massgebende Richtung
M_ok = H_k·(H_Kopf+ü) + M_G + M_W         [kNm] Moment an OK Block (Bezugspunkt)
3.4.3 · Kennzeichnende Drehwinkel (Gl.11–12)
tan α₁ = 6·(N_α·tan φ₂ + H_k) / (B·D²·C₁)    (Gl.11) Überwindung Sohlreibung
tan α₂ = 2·N_α / (A²·B·C₂)                    (Gl.12) Abheben hintere Sohlkante
C₂ = C₁ (vertikale = horizontale Bettungsziffer nach Steckner)
